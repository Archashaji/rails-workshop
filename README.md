# Rails workshop

We created this repo to make our lives easier when doing ruby on rails workshops.

We have followed Michael Hartl's ruby on rails tutorial.

Please feel free to use this for conducting workshops

## Usage

 - Clone this repo
 - do a `bundle install`
 - run `rails server`
 - enjoy
